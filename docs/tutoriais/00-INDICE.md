# 📚 Tutoriais de Uso — CM.segCRM

Bem-vindo(a) ao guia oficial de utilização do **CM.segCRM**, o sistema de gestão de seguros da CM Seguros.

Este material foi preparado para ser **apresentado a todos os usuários da equipe** — corretores, líderes, financeiro, RH e administradores. Cada arquivo aborda um conjunto de módulos, com objetivo, telas, campos, permissões e o passo a passo de uso.

---

## Como ler este guia

Cada tutorial segue a mesma estrutura:

1. **Para que serve** — em uma linha, o que o módulo resolve
2. **Onde fica** — caminho no menu lateral
3. **Quem pode usar** — perfis com acesso (Admin, Líder, Corretor, Financeiro, RH, Pós-venda, Gestão)
4. **Visão geral da tela** — o que aparece quando você abre
5. **Passo a passo** — fluxo prático de uso
6. **Dicas e atalhos** — boas práticas e cuidados

## Glossário rápido

| Termo | Significado |
|---|---|
| **Negócio** | Cada oportunidade de venda dentro de um funil (cliente + produto + valor + etapa) |
| **Funil** | Processo de vendas organizado em etapas (ex: VENDA, RENOVAÇÃO, COBRANÇA) |
| **Apólice** | Contrato de seguro emitido após o negócio ser ganho |
| **Proposta** | Documento da seguradora antes da apólice ser emitida |
| **Etapa** | Coluna do kanban dentro de um funil (ex: "Cotação enviada", "Em análise") |
| **Vendedor / Corretor** | Pessoa responsável pelo negócio dentro do CRM |
| **Líder** | Responsável por uma equipe de corretores |
| **Admin** | Acesso total ao sistema |
